# Basic ray tracer on native JavaScript.

## Supports the following functions:

- [x] Drawing primitives;
  - [x] Spheres;
  - [ ] Polygons;
- [x] Lighting rendering:
  - [x] Point light;
  - [x] Directional light;
  - [x] Ambient light.
- [x] Diffusion reflection;
- [x] Specular reflection;
- [x] Shadows;
- [x] Reflections;
